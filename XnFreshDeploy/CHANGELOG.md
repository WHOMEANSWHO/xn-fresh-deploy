# Changelog

## 3.10 - 2026-07-14

- Added dedicated custom-command sections to profile creation and editing with separate validated name/value fields.
- Custom commands appear as selected picker entries, can override a built-in value, can be removed individually, and remain covered by profile backups and shortcuts.

## 3.9 - 2026-07-14

- Replaced free-text profile commands with a clear built-in multi-select command picker: click one command or Ctrl+click several.
- Added value fields for mouse scale and first-person FOV, mutually exclusive crosshair choices, exact acceleration-command variants, and preservation of older custom commands.

## 3.8 - 2026-07-14

- Keeps FiveM on the official Latest (Unstable) Canary update channel and verifies that setting before profile launches.
- Adds validated per-profile FiveM session settings, with create/edit support, session-isolated launches, profile-card counts, and export/import coverage.

## 3.7 - 2026-07-14

- Added the dedicated ReShade manager with verified install, update, repair, DLL switching, health checks, shader choices, preset compatibility checks, acknowledgement handling, and manifest-based uninstall.
- Added crash recovery, exact ReShade rollback snapshots, full portable backups, readiness badges, safe import previews, cached verification, automatic shortcut maintenance, and one-click setup rollback.
- Added profile editing, duplication, favourites, last played, search, sorting, connection testing, and FiveM server detection.
- Added atomic profile saves, clearer first-run errors, safer optional-app defaults, validated configuration, a hardened launcher path, and cached ReShade health metadata for distribution use.
