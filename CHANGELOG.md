# Changelog

## 4.1.0 — 2026-07-15

### Added
- Native WPF desktop app (no PowerShell, no separate .NET install)
- Server profiles with soundpack, ReShade look, and startup commands
- Pack library with drag-and-drop import and safety preview
- Guided PC setup with elevated background worker
- First-run walkthrough
- Toast notifications
- Keyboard shortcuts: Ctrl+N, Ctrl+F, Enter to play
- Profile folders and tags with filtering
- FiveM install/running status badge
- Auto-update check via GitHub Releases
- Legacy import from older XnFreshDeploy folders
- Crash logging to `logs/`
- Window size and position memory
- Portable release ZIP with SHA256 checksums

### Changed
- Redesigned UI with warmer charcoal theme and cleaner layout
- Profile cards simplified: Play, Edit, More menu

## Earlier versions

Versions 3.x and below used the PowerShell-based `Xn-Setup.ps1` workflow.
